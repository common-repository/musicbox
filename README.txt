Add custom playlist widgets to your website including preview clips from iTunes.

== Description ==

This plugin makes it simple to add custom music playlists including:

* Preview from iTunes
* Artwork
* Artist Name
* Song Title
* Album or Collection
* Links to iTunes

Unlimited Musicboxes and tracks. Drag and drop re-ordering of tracks and Musicboxes. You can set the widget to play automatically or require the user to press play first. The widget will play through all of the tracks.

__PHP 5 >= PHP 5.3.0 or  Required__

<a href="http://www.webdesignby.com/wordpress/plugins/musicbox/">Learn More About Musicbox</a>

== Installation ==

1. Upload `webdesignby-musicbox` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

1. Click the new Musicbox menu item in wp-admin
2. Create a new musicbox
3. Add tracks to the musicbox
4. Repeat as necessary
5. Add music box to pages using the Musicbox widget
6. Add music box to pages using the Musicbox shortcode 

Shortcode ex. 
[musicbox musicbox_id="1" description="1" autoplay="1" tracks_perpage="2"]
